This project is my personal portfolio website where I present my professional profile, technical skills, and real projects in a clean and responsive interface.

Built with **Astro** and **Tailwind CSS**, the site focuses on speed, accessibility, and a clear user experience across desktop and mobile devices.

## Features

- Responsive layout and smooth navigation
- About, experience, projects, and contact sections
- Reusable Astro components
- SEO-friendly structure
- Easy content updates through markdown and JSON data

## Technologies Used

- Astro
- Tailwind CSS
- HTML5
- JavaScript
